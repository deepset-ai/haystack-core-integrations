## Installation

```console
pip install google-ai-haystack
```


## Contributing

`hatch` is the best way to interact with this project, to install it:

```sh
pip install hatch
```

With `hatch` installed, to run all the tests:

```
hatch run test
```

To run the linters `ruff` and `mypy`:

```
hatch run lint:all
```
