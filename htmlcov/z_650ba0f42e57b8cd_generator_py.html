48 :param api_key: The Together API key. 

49 :param model: The name of the model to use. 

50 :param api_base_url: The base URL of the Together AI API. 

51 :param streaming_callback: A callback function that is called when a new token is received from the stream. 

52 The callback function accepts StreamingChunk as an argument. 

53 :param system_prompt: The system prompt to use for text generation. If not provided, the system prompt is 

54 omitted, and the default system prompt of the model is used. 

55 :param generation_kwargs: Other parameters to use for the model. These parameters are all sent directly to 

56 the Together AI endpoint. See Together AI 

57 [documentation](https://docs.together.ai/reference/chat-completions-1) for more details. 

58 Some of the supported parameters: 

59 - `max_tokens`: The maximum number of tokens the output text can have. 

60 - `temperature`: What sampling temperature to use. Higher values mean the model will take more risks. 

61 Try 0.9 for more creative applications and 0 (argmax sampling) for ones with a well-defined answer. 

62 - `top_p`: An alternative to sampling with temperature, called nucleus sampling, where the model 

63 considers the results of the tokens with top_p probability mass. So, 0.1 means only the tokens 

64 comprising the top 10% probability mass are considered. 

65 - `n`: How many completions to generate for each prompt. For example, if the LLM gets 3 prompts and n is 2, 

66 it will generate two completions for each of the three prompts, ending up with 6 completions in total. 

67 - `stop`: One or more sequences after which the LLM should stop generating tokens. 

68 - `presence_penalty`: What penalty to apply if a token is already present at all. Bigger values mean 

69 the model will be less likely to repeat the same token in the text. 

70 - `frequency_penalty`: What penalty to apply if a token has already been generated in the text. 

71 Bigger values mean the model will be less likely to repeat the same token in the text. 

72 - `logit_bias`: Add a logit bias to specific tokens. The keys of the dictionary are tokens, and the 

73 values are the bias to add to that token. 

74 :param timeout: 

75 Timeout for together.ai Client calls, if not set it is inferred from the `OPENAI_TIMEOUT` environment 

76 variable or set to 30. 

77 :param max_retries: 

78 Maximum retries to establish contact with Together AI if it returns an internal error, if not set it is 

79 inferred from the `OPENAI_MAX_RETRIES` environment variable or set to 5.
