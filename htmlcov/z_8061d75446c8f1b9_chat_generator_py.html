16@component 

17class EdenAIChatGenerator(OpenAIChatGenerator): 

18 """ 

19 A chat generator that uses Eden AI's OpenAI-compatible API to generate chat responses. 

20 

21 Eden AI is a unified API that gives access to 500+ AI models from many providers (OpenAI, 

22 Anthropic, Mistral, Google, Cohere, and more) through a single API key, with built-in 

23 provider fallback and EU data residency. This makes it a convenient, sovereignty-friendly 

24 gateway for building LLM and RAG applications with Haystack. 

25 

26 This class extends Haystack's `OpenAIChatGenerator` to talk to Eden AI. It sets the 

27 `api_base_url` to Eden AI's OpenAI-compatible endpoint and keeps all the standard 

28 configurations available in the `OpenAIChatGenerator`. 

29 

30 Models are selected using Eden AI's `provider/model` naming convention, for example 

31 `"openai/gpt-4o-mini"`, `"anthropic/claude-sonnet-4-5"`, or `"mistral/mistral-large-latest"`. 

32 See the [Eden AI models catalog](https://www.edenai.co/models) for the full list. 

33 

34 Usage example: 

35 ```python 

36 from haystack_integrations.components.generators.edenai import EdenAIChatGenerator 

37 from haystack.dataclasses import ChatMessage 

38 

39 messages = [ChatMessage.from_user("What's Natural Language Processing?")] 

40 

41 client = EdenAIChatGenerator(model="mistral/mistral-large-latest") 

42 response = client.run(messages) 

43 print(response["replies"][0].text) 

44 ``` 

45 """

67 :param generation_kwargs: Optional keyword arguments passed to the underlying generation API call,
